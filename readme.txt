== Description ==
= About Kit.co (formerly kit.com) =
Kit is a community to discover, discuss and get interesting products – grouped into kits – for activities like traveling, DJing, cooking, cycling and more. To learn more about Kit, please click [here](https://kit.co/for-creators).

= About Kit Wordpress Plugin =
kit.co WordPress Plugin is a free plugin for WordPress. The plugin enables you to paste links to your kits on [kit.co](https://kit.co) and generate interactive, shoppable modules of your favorite products that you recommend, directly on your WordPress blog.

= Note =
By installing this plugin you agree to accept kit.co’s [Terms of Use](https://kit.co/terms).


== Installation ==
= Sign up for kit.co =
Sign up for a free account at [kit.co](https://kit.co). Create kits of your favorite products that you use and recommend.

= Automatic Installation =
1. Log in to your WordPress dashboard, navigate to the Plugins menu and click Add New.
2. Search for "kit.com". Click "Install Now".

= Manual Installation =
If you are self-hosting your WordPress blog you can use the following instructions:

1. Make a 'kit-com' directory in '/wp-content/plugins/'.
2. Upload all files into the
'/wp-content/plugins/kit-co/' directory.
3. Activate the plugin through the 'Plugins' menu in WordPress.

== Frequently Asked Questions ==
= Where can I find more information on kit.co? =
You can find out more on [kit.co](https://kit.co) and you can find answers to many frequently asked questions on our [FAQ](https://blog.kit.co/got-questions-8368a2eb3526).

= How does the plugin work? =
kit.co supports the oEmbed protocol ([http://oembed.com/](http://oembed.com/)) for allowing embedded representations of kit urls on third party sites. This plugin uses the wordpress function [wp_oembed_add_provider](https://codex.wordpress.org/Function_Reference/wp_oembed_add_provider) to register kit.co as an oEmbed provider for your blog, allowing all users who can write posts to paste kit.co links and have them automatically replaced with interactive embed modules in the body of a post or page.

= How much do Kit and this plugin cost to use? =
It is free to create an account on kit.co as well as to use the WordPress plugin.
